Installing
----------

Manual install Python 2:
~~~~~~~~~~~~~~~~~~~~~~~~

``cd`` to the library directory, and run:

.. code:: bash

    sudo python setup.py install

Manual install Python 3:
~~~~~~~~~~~~~~~~~~~~~~~~

``cd`` to the library directory, and run:

.. code:: bash

    sudo python3 setup.py install
